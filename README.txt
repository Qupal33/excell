Финансы продаж АЦБК

Локальное приложение для формирования Excel-отчётов отдела продаж.

БЫСТРЫЙ ЗАПУСК

Для обычного запуска откройте файл:
- Финансы продаж АЦБК.exe

Файл лежит в корне проекта и запускается двойным кликом.

ВОЗМОЖНОСТИ

- выбор исходных Excel-файлов
- формирование итогового отчёта
- сохранение результата в формате .xlsx

ЗАПУСК ДЛЯ РАЗРАБОТКИ

Нужен установленный Node.js.

Команда:
- npm.cmd start

СБОРКА EXE

Команда:
- npm.cmd run dist

После сборки portable-файл появляется в папке:
- dist

Имя файла:
- finance-sales-reports-0.1.0.exe

При необходимости его можно скопировать в корень проекта и назвать:
- Финансы продаж АЦБК.exe

ИКОНКА ПРИЛОЖЕНИЯ

Файлы иконки:
- build/icon.ico
- build/icon.png

Скрипт генерации:
- build/generate-icon.ps1
