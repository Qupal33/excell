Финансы продаж АЦБК

Локальное приложение для формирования Excel-отчётов отдела продаж.

Запуск:
1. Открыть папку проекта.
2. Выполнить команду: npm.cmd start


1. dist/win-unpacked
2. файл Финансы продаж АЦБК.exe

Сборка:
1. Выполнить команду: npm.cmd run dist:dir

Поддерживаемые входные файлы:
- .xlsx
- .xlsm
- .xls

Результат:
- итоговый отчёт сохраняется в формате .xlsx
