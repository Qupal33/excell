# excell

Простое Electron-приложение для формирования Excel-отчётов.

## Сборка EXE

Требуется установленный `Node.js`.

```bash
npm install
npm run dist
```

После сборки `.exe` появится в папке `dist`.

## Возможности

- выбор исходных Excel-файлов;
- формирование итогового отчёта;
- подсчёт общего количества созданных отчётов.
