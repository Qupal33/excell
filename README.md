# Финансы продаж АЦБК

Локальное Electron-приложение для формирования Excel-отчётов отдела продаж.

## Быстрый запуск

Для обычной работы используйте файл:

- `Финансы продаж АЦБК.exe`

Он находится в корне проекта и запускается двойным кликом.

## Возможности

- выбор исходных Excel-файлов;
- формирование итогового отчёта;
- сохранение результата в формате `.xlsx`;
- локальная работа без браузера.

## Запуск для разработки

Требуется установленный `Node.js`.

```bash
npm install
npm start
```

## Сборка EXE

Чтобы пересобрать приложение в `.exe`, выполните:

```bash
npm run dist
```

После сборки portable-файл появится в папке `dist`:

- `dist/finance-sales-reports-0.1.0.exe`

При необходимости его можно скопировать в корень проекта и переименовать в:

- `Финансы продаж АЦБК.exe`

## Иконка приложения

Файлы иконки лежат в папке `build`:

- `build/icon.ico`
- `build/icon.png`

Скрипт генерации иконки:

- `build/generate-icon.ps1`
